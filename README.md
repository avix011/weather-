🌦 Advanced Weather Dashboard
A modern, fully responsive Weather Dashboard Web Application built using HTML, CSS, and JavaScript. This project provides real-time weather updates, forecasts, and interactive visual features with a clean and attractive UI.

🚀 Features

✨ Real-Time Weather Data
Get current weather details of any city worldwide
Temperature, humidity, wind speed, and pressure

📅 7-Day Forecast
Displays upcoming weather conditions
Daily temperature and weather icons

📍 Location-Based Weather
Auto-detect user location using Geolocation API

🎨 Modern UI & Animations
Glassmorphism design
Smooth transitions and animated background

🌙 Dark / Light Mode
Toggle between themes for better user experience

🗺 Live Weather Map
Interactive map using Leaflet.js
Shows real-time weather layers

🌧 Weather Animations
Rain, snow, clouds, and lightning effects

💾 Responsive Design
Works on mobile, tablet, and desktop

🛠 Tech Stack
HTML5 – Structure
CSS3 – Styling & Animations
JavaScript (ES6) – Functionality
OpenWeather API – Weather data
